Printer exploitation

This section will touch on targeting printers while on the same network. Looking to strictly fuck with printers on the network? Planning on using the network printer to print a bunch of ASCII dick images or a blackmail page? Well you're in luck kid!

In Kali open a new terminal window:
git clone https://github.com/RUB-NDS/PRET.git
pip install colorama pysnmp
cd PRET
./pret.py


If you encounter a printer that you're able to exploit it will be displayed here. You might actually come across some printers in the wild that have their own open access points that you're able to connect to. If you are able to connect to an open printer network you might want to run this attack framework to see if there's anything useful you're able to do against the printer. Might be your foot in the door onto a network. Either than that this is kind of stupid but fun.

For more information and documentation on the commands please refer to:
https://github.com/RUB-NDS/PRET
