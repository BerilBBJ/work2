Router exploitation

This chapter really isn't that useful but some people might find it helpful. There may be times when you want to exploit the router for whatever reason. It's doubtful you'd want to do this but maybe you just want to watch the world burn so you go around hacking other people's Wi-Fi networks just so you can corrupt the routers firmware and fuck it all up for everyone. You fucking maniac.

Whatever your reason is there are router exploit frameworks already in place to help with your motives.

Let's download one of the frameworks that can assist with attacking the router directly.

In Kali open a new terminal window:
git clone https://www.github.com/threat9/routersploit
cd routersploit
python3 -m pip install -r requirements.txt
python3 rsf.py

If you're familiar with Metasploit the syntax is pretty much the same in regards to navigating around. Honestly, the only thing you need to know about this framework is the autopwn function unless you're super into hacking firmware and other shit. Which your not.

While using routersploit:
use scanners/autopwn
show options
set target ROUTER_IP
run



This will run through every public exploit that's available for the specific router targeted. If routersploit finds something vulnerable you can then go to the exploits section (use exploits/exploit same as Metasploit ish) and launch the corresponding exploit. Super easy to use and shouldn't be overlooked if your goals are to go full retard on the router. Why hack people right? Once you've gained access you'll be presented with a command line on the router so you're able to navigate, delete, replace firmware, and alter files accordingly on the router.
